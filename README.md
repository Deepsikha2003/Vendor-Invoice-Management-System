# Vendor & Invoice Management System
A full-stack web application designed to manage vendors and track invoices efficiently. The system allows users to create, update, and monitor vendor details and invoice records, including payment status and due dates.
It provides a centralized dashboard to view financial summaries such as total invoices, paid/unpaid amounts, and overdue payments. The application helps streamline business operations by organizing vendor interactions and improving invoice tracking.
Built using HTML, CSS, JavaScript, Node.js, Express, and MongoDB.
